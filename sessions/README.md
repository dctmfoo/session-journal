# Session journals for this repository

This project dogfoods its own specification. Journals use root `sessions/`, UTC labels in committed examples, and THIN mode against repository issues or `docs/verification.md` when a durable artifact governs the work. Evidence vocabulary includes bats suite names, runtime versions, workflow run URLs, and commit hashes.

Follow `spec/session-journal-spec.md` and start from `templates/session-journal.md`. Never include credentials or private environment details.
